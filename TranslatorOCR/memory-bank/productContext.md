# Product Context

Why this project exists:
- Provide reliable in-game subtitle translation with minimal resource impact and long-term maintainability.

Problems it solves:
- Real-time translation of on-screen text (game subtitles) without modifying the game.
- Centralizes OCR, preprocessing, and translation logic in a testable .NET service.

User experience goals:
- One-button region selection and start/stop control.
- Non-intrusive, draggable overlay that displays translated text.
- Simple settings for tessdata location and OCR language.
